üé® Colors

CSS supports various color formats:

  • Hexadecimal: #RRGGBB (e.g., #ff0000)
  • RGB: rgb(255, 0, 0)
  • RGBA: With opacity. rgba(255, 0, 0, 0.5)
  • HSL: hsl(0, 100%, 50%) (Hue, Saturation, Lightness)
  • Named Colors: red, blue, green, teal, etc.

üìè Units

Width: 200px (absolute)
Width: 50% (relative to parent)
Font Size: 2em (relative to parent)
Font Size: 2rem (relative to root)
Height: 20vh (viewport height)
Width: 20vw (viewport width)
Width: 20ch (20 characters)
Font Size: 2ex (height of 'x')
Best Practice: Use rem for scalable typography and vh/vw for full screen layouts.

üì¶ Margin & Padding

Margin (Outer space)

Shorthand

  • margin: 10px; ‚Üí all sides
  • margin: 10px 20px; ‚Üí top/bottom 10px, left/right 20px
  • margin: 10px 15px 20px 25px; ‚Üí top, right, bottom, left

Margin Collapse

Two vertical margins of adjacent blocks may collapse into one. Example: Two <p> tags each with margin-top: 20px; may result in 20px, not 40px.

üìå Summary

  • Colors: Support multiple formats like Hex, RGB, HSL, and named colors.
  • Fonts: Define font-family, weight, size, style, and line-height.
  • Units: Understand absolute (px) vs relative (%, em, rem, vw, vh).
  • Margins: Space outside the element. May collapse.
  • Padding: Space inside the element.
