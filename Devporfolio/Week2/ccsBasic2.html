1. The CSS Box Model

All HTML elements are considered boxes in CSS. The box model includes:

  • Content: the actual text or image
  • Padding: space around the content
  • Border: edge around the padding
  • Margin: space outside the border

3. CSS Specificity

Specificity determines which CSS rule is applied when multiple rules match the same element.

Order of Specificity

  1. Inline styles (e.g. style="...")
  2. ID selectors (#id)
  3. Class, attributes, and pseudo-classes (.class, [type], :hover)
  4. Element and pseudo-elements (div, p, ::before)

!important overrides all other declarations (use carefully!)
