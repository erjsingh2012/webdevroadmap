CSS Box Model

The CSS box model is a fundamental concept. Each HTML element is a box composed of the following:

  • Content: The actual content of the box (text, image, etc.)
  • Padding: Space around the content
  • Border: Wraps the padding and content
  • Margin: Space outside the border
Box Example 

.box-example {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid #00796b;
  margin: 20px;
  background: #e0f7fa;
}
